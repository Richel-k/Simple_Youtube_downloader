ℹ️ Comment fonctionne le téléchargement en détail ?

Cette application utilise le moteur yt-dlp pour extraire directement les flux serveurs de YouTube. Le processus se déroule en deux étapes clés :

  1. L'analyse des flux (Étape 1) : En collant votre lien et en cliquant sur Analyser, le serveur interroge la vidéo. YouTube sépare presque toujours les flux de haute qualité en deux fichiers distincts : un fichier pour l'image seule, et un fichier pour le son seul.
  2. Le téléchargement ciblé (Étape 2) : La console verte vous affiche la liste de ces flux. Repérez la colonne VCODEC et ACODEC pour faire votre choix, puis renseignez l'ID correspondant.
⚠️ Attention importante concernant les formats "video only" :

Si vous choisissez un ID de haute résolution (comme 1080p ou 720p), vous remarquerez souvent la mention video only dans la description du format. Si vous téléchargez cet ID seul, votre vidéo finale n'aura aucun son.

La solution : Vous devez combiner l'ID de la vidéo avec un ID de flux audio (marqué audio only, comme l'ID 140 ou 251).

Dans le champ de saisie de l'ID, liez-les simplement avec le symbole +.
Exemple : Écrire 136+140 va télécharger la vidéo HD (136) ET l'audio (140). L'application utilisera ensuite FFmpeg en arrière-plan pour fusionner automatiquement les deux en un seul fichier complet et propre.
